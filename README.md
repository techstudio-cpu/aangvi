# AANGVI – Full Production System

This repository contains the COMPLETE production-ready codebase for the AANGVI
Wedding Planning & Event Management Platform.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Prisma + PostgreSQL (Railway)
- Sanity CMS
- Cloudinary
- WhatsApp Cloud API
- Resend Email
- NextAuth (Email/Password)
- Tailwind CSS
- Framer Motion

## Features
- CMS-driven content (services, weddings)
- Admin dashboard with role-based access
- Lead capture with DB + WhatsApp + Email
- Cloudinary image optimization
- ISR & caching
- Railway deployment ready

## Setup
```bash
npm install
npx prisma migrate dev
npm run dev
```

## Deployment
Optimized for Railway (standalone output).
